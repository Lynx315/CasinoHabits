# 🎰 Slot Machine Habit System

A gamified habit-tracking application that transforms mundane habit completion into an exciting casino experience. Built with Python and Streamlit.

### Live Application
Visit the app here: **[https://casinohabits.streamlit.app/](https://casinohabits.streamlit.app/)**

### Features
- **Habit Setup**: Configure your habit, clip values, and custom reward tiers
- **The Jar**: Draw colored clips each time you complete your habit
- **Tier System**: Cash in clips to unlock reward tiers (Tier 1, 2, 3, or Jackpot)
- **Main Wheel**: Spin the wheel to win your rewards with a dynamic, animated experience
- **Save/Load**: Download and load your progress using the Casino Memory Card system
- **The Naked Rule**: Never enjoy these rewards outside the system — only through the game!

### Setup Your Casino
1. Define your habit (e.g., "15 Burpees")
2. Set your clip value ($)
3. Configure your rewards for each tier and the jackpot
4. Start drawing clips and building momentum!

### How It Works
1. **Draw Clips**: Complete your habit → draw a random colored clip
2. **Cash In**: Collect 1+ clips of a color to unlock reward tiers
3. **Spin**: Activate the main wheel to win your rewards
4. **Repeat**: Keep the momentum going!
